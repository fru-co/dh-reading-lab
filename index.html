使い方 / How To Use

  • 作品を選ぶ（またはテキストを貼り付ける）
  • 1つの語を選んで検索する
  • 頻度・文脈・分布を見る

分析ツール / Tools

  • 頻出語: よく出る語を数える
  • KWIC (Key Word in Context): 語の前後の文脈を見る
  • 分布: 語がどこに現れるか見る
